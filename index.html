Ingenieria de software


La Ingeniería de Software es una de las ramas de las ciencias de la computación que estudia la creación de software confiable y de calidad, basándose en métodos y técnicas de ingeniería. Brindando soporte operacional y de mantenimiento, el campo de estudio de la ingeniería de software. ​Integra ciencias de la computación, ciencias aplicadas y las ciencias básicas en las cuales se encuentra apoyada la ingeniería.

User stories

¿Qué son las historias de usuario?
Las historias de usuario, son descripciones cortas de una necesidad de un cliente del software que estemos desarrollando.
Las historias de usuario en 5 pasos:

  1. Identifica fuentes de información
  2. Recopilar la información seleccionando una técnica
  3. Escribir las historias (en una tarjeta):
  4. Revisar / Conversar sobre las historias de usuario
  5. Ampliar y refinar las historias de usuario

TDD

El desarrollo dirigido por tests es una técnica basada en realizar pequeños tests que describen la funcionalidad antes de desarrollarla.

Tests de integración

Prueban la integración entre dos o más componentes.

Test funcionales

Sirven para probar la aplicación desde el punto de vista del usuario final.
